# Port Scanner (Python):

# Overview:

A simple Python-based **port scanner** that detects open ports on a target system using socket programming and multithreading.



# Features:

* Scan ports (1–1024)
* Detect open ports
* Supports domain or IP
* Fast scanning using threads



# Tech Stack:

* Python
* Socket programming
* Multithreading




# Disclaimer:

This tool is for **educational purposes only**.
Only scan systems you own or have permission to test.
